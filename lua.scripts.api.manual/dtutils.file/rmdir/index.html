🔗NAME

rmdir

🔗SYNOPSIS

recursively remove a directory

🔗USAGE

local df = require "lib/dtutils.file"
df.rmdir(path)

path - string - a directory path

🔗DESCRIPTION

rmdir allows directories to be removed recursively.

🔗RETURN VALUE

path - string - a directory path
